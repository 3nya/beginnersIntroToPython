What is Coding?

Coding, also known as programming, is the process of creating instructions for computers to execute. It involves writing sequences of commands using a specific programming language to perform various tasks and solve problems.

What is Computer Science?

The theory, design, development, and application of computers and computational systems are all studied in computer science. It covers a broad range of subjects, including artificial intelligence, software engineering, data structures, algorithms, and more.

One of the core concepts of computer science is coding. It is the practical application of theoretical ideas to the development of systems, software, and algorithms that make it possible for computers to carry out tasks effectively. Coding is a major tool used in computer science to implement algorithms, create software, and solve challenging issues in a variety of fields.

Coding is a great way to develop logical reasoning and problem-solving abilities. It makes it possible for anyone to create algorithms that effectively tackle complicated issues by dissecting them into smaller, more manageable jobs.

Why Should I Learn to Code?

Empowerment: Learning to code gives people the ability to solve difficulties in their lives in a technologically advanced world. Whether developing a mobile app, or website or automating tedious processes, it enables users to bring their ideas to life.

Career opportunities: Knowing how to code gives up a lot of options for careers across many different industries. Professional programmers are in high demand in a variety of industries, including web development, data science, artificial intelligence, and software development.

Innovation: The ability to code allows people to create new tools, software, and technologies that promote advancement and change in the community. Innovations in industries like healthcare, banking, transportation, and entertainment are centered around it.

Possible Career Paths with Coding

There are many different career paths available to coders. Here are some inspiring professions:

Web developers use programming languages like HTML, CSS, and JavaScript to create websites and web apps.

Game Developers use programming knowledge and creativity to design and create video games for consoles, PCs, and mobile devices.

Cybersecurity Analysts create security solutions and put data and information safety measures in place to protect computer systems and networks from cyber threats.

Installation[e][f]

In this section of the Basics of Python book, we’ll guide you through the process of installing IDLE onto your computer, whether you have a Windows or Mac. Before we get into how to install IDLE, please take a look at the vocabulary words below.

Vocabulary Words

  1. IDLE: Also known as Integrate Development and Learning Environment. This is a software application programmers use for software development. Think of this as your own online notebook where you can use code to command the computer various tasks along with some fancy tools (source code editor, build automation tools, and a debugger).  
  2. Dependency: A dependency is a software component that another software module requires to function correctly. Think about it as you need a piece of bread in order to make a PB and J sandwich.
  3. Binary Distribution: A binary distribution is a precompiled version of software that is ready for installation on a specific platform without requiring further compilation. When installing IDLE, you'll typically download a binary distribution tailored for your operating system.

Downloading and Installing IDLE

To download and install IDLE on your Windows or Mac operating system, follow these steps:

  1. Open your web browser (like Google, Safari, Firefox) and search up “official Python website” or use this link:  https://www.python.org/.
  2. On this website, select the “Downloads” section. Here you will find the latest version of Python available for download.
  3. Scroll down the page and select the installer for your computer. For Windows, you'll typically choose the executable installer (ending in ".exe"), while for Mac, you'll select the macOS installer (ending in ".dmg").
  4. Once the installer is downloaded, double-click on it to run the installation process. Follow the prompts to install.
  5. While you install, you may have the choice to customize your installation. Customize to your preferences.
  6. After customizing your installation options, continue with the installation by clicking "Install" or "Finish," depending on your operating system. The installer will then install Python and IDLE on your computer.
  7. Once the installation is complete, you can check that it has been installed by searching for it in your computer’s applications or programs menu. You can also open a command prompt or terminal and type idle to launch IDLE.

Setting up IDLE on Windows and Mac

After installing IDLE, you may need to set it up to ensure it works smoothly on your Windows or Mac system. Follow these steps to set up IDLE:

  1. Open IDLE by searching for it in your applications or programs menu. Once opened, navigate to the "Options" or "Preferences" menu to configure IDLE settings according to your preferences. You can customize settings such as font size, color scheme, and indentation preferences.
  2. After configuring preferences, test IDLE by opening a new Python file or interactive shell. Write a simple Python script or execute some Python commands to ensure that IDLE is functioning correctly.
  3. Take some time to explore the various features of IDLE, such as the integrated code editor, debugger, and interactive shell. Be familiar with the layout and how the IDE works to be productive in your Python journey!

Tip: You can customize! You can adjust settings such as font size, color scheme, and indentation preferences in the Options or Preferences menu. Do whatever will make you feel excited and comfortable about coding!

Introduction to Data in Python

Data refers to facts, statistics, or information collected for analysis or reference. Data is the building blocks of programs and computational tasks in Python. Data can take many forms, including:

Order of Operations

Just like PEMDAS in math, there is an order of operations in Python. In other words, when you have multiple operations in one statement, they will be executed in a specific sequence.

Tip: If you want a certain operation to go before other ones, put parentheses around them. The operations inside parentheses run first.

The order of operations in Python is basically the same as in math. Going left to right, the order is first parentheses (()), then exponentiation (**), then multiplication/division/truncated division/modulus (*, /, //, %), and finally addition/subtraction (+, -).

Later in this book, you will learn about non-arithmetic operators. However, these operations have less precedence than the ones you have learned so far and would be executed after.

SECTION 7 - Debugging

It’s okay to have errors within your code. But it’s not okay to leave them unfixed. This section will go through common errors within your program and how to fix them!

7.1
Types of Errors

Vocabulary Words

  1. Syntax Error: an error when code doesn’t follow Python language syntax rules (ex: incorrect indentation, misspelling keywords, etc)
  2. Runtime Error: an error when an unexpected condition prevents the program from continuing when being executed. (ex: dividing by zero)
  3. Type Error: a type of runtime error when an operation is performed on an incompatible type (ex: performing arithmetic operations on strings)

Errors

When creating your own computer programs, you are bound to run into many errors. But don’t worry, errors are normal to face when coding. Error messages provide important information on the type of error and the specific lines in the program where the error was found. This information is important in debugging, as different errors lead to different approaches to fix the error.

Syntax Error

Syntax errors are one of the most common errors seen, especially for beginners, but easy to fix. They occur when your code doesn’t follow Python syntax. Oftentimes, the error message shows the specific line where the error occurred, so look back to that line to make sure the syntax is correct. Keep an eye out for indentation, use of parentheses/quotes, or misuse of operators.

Runtime Error

Runtime errors occur when an unexpected condition is detected which prevents the execution of the program. Examples include dividing by zero and misspelled variables and function names.

Type Error

Type errors occur when an operation is performed on an inappropriate variable or data type. This includes mathematical or logical operations being performed on strings, or attempting to convert a variable to an incompatible type (ex: string to int).

Tip: Always look at the Python error message found when you run your program, it really helps figuring out how to approach the error, what type of error, and where the error occurred.

You just finished typing up your code, but when you clicked the button, the program didn't work! How to Debug

All errors are not the same. There are 2 different kinds of errors with examples!
